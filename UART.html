Configurer ou administrer un serveur Linux grâce à l'UART

L'UART est une connexion USB entre le module de developpement (OPI PC +) et un ordinateur pour avoir accès au terminal de commande.


Avantages :

- Pratique si aucune configuration n'est présente sur la carte. Soit à l'initialisation pour configurer le réseau, soit lorsque le réseau à été perdu.

- Parfait pour éviter l'utilisation d'un écran, d'un clavier et d'une souris supplémentaire.


Désavantages :

-Contrairement au SSH, cette methode ne permet pas l'accès distant et necessite une proximité immediate du module et de l'oppérateur.



Préparation du module de developpment (OPI PC+)

1. Installer une distribution Linux sur Orange Pi PC + comme décrit ici

2. Relier les pins du cable UART sur la carte comme indiqué que la photo.


- Le cable Noir est le GND donc la masse, il faut le brancher sur le GND de la carte.
- Le cable vert est le TX, il doit donc etre branché sur le RX de la carte.
- Le cable blanc est le RX, il doit donc etre branché sur le TX de la carte.

Préparer Windows 10

3. Brancher le cable UART à l'ordinateur. Il doit signaler par un son le branchement d'un nouveau peripherique. W10 l'installe mais pas correctement...
Installez donc le pilote suivant : Pilotes UART pour w10.
Rendez-vous dans le Gestionnaire de péripheriques et mettre à jour le péripherique manuellement comme ci dessous :



Les pilotes sont installés, Windows 10 est donc prêt à fonctionner avec la liaison UART.



Administrer via UART grace à Putty

4. Télécharger puis installer Putty, qui permet d'avoir accès au terminal de commande de l'OPI PC +.

lancer le gestionnaire de péripheriques et se rendre dans l'encart "Ports(COM et LPT)" pour connaitre le numéro du port attribué au cable UART. Ici dans l'exemble il s'agit du port "(COM3)"


Ouvrir Puty et configurer comme ci dessous :

Serial line : (COMx) comme indiqué dans le gestionnaire de péripheriques

Speed : 115200

Connexion type : Serial

Cliquez sur OPEN et le teminal se lance!
